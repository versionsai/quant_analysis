# OpenAPI Skills 入口

本目录用于存放外部行情/交易 OpenAPI 的协作型 Skills 文档。

当前已接入：
- `../futu/openapi.md`：Futu OpenAPI / OpenD 接入与验证说明

建议使用方式：
- 官方能力说明优先参考对应 OpenAPI 官方文档
- 项目内 `skills/` 文档负责补充本项目自己的配置约定、运行路径和排障经验

建议放置内容：
- 外部 API 接入步骤
- 本地/局域网配置约定
- 行情权限与接口能力说明
- 故障排查手册

约束：
- 本目录内容默认不会被项目运行时代码直接加载
- 仅用于指导开发助手或维护者执行操作
